## 個人情報保護方針

当個人情報保護方針は、本サイトや、その他の私が制作・運営するアプリケーションや情報を使用することにより収集された情報を対象としています。

#### 私は、配布するアプリケーションを通じて子供から個人情報を収集することは一切ありません。

あなたが、私が配布するアプリケーションをダウンロードして使用する際、私はアプリケーションの使用時間や、その画面表示回数もしくは作業回数等の使用方法に関する非個人的な情報を除き、あなた又はあなたのデバイスに関する情報を収集することは一切ありません。

#### 私は、社外のいかなる者に対しても任意の非個人的情報を提供することはありません。

#### 私は、児童オンラインプライバシー保護法 を順守します。

私が制作・運営するウェブサイト及び子ども向けアプリケーションは、児童オンラインプライバシー保護法（「COPPA」）に準拠しています。当社は故意に13歳未満の子供から個人情報を収集することはせず、もしサポートリクエストもしくは何らかのフィードバックを通じてユーザーが13歳未満であることが判明した場合、私はそのようなユーザーの一切の個人情報の収集、保管および使用を行わず、安全な方法で消去します。

ユーザーが私が制作・運営するアプリケーションの使用に関するサポートを求める場合、ユーザーは私と連絡をとることができます。あなたがサポート要求を送信する場合、私はあなたのEメールアドレス及びサポート要求に関してあなたが提供したその他の情報（サポート情報）を受信するかもしれません。当社はサポート情報のみ使用します。

#### 情報の共有と開示

私はあなたのEメールアドレス情報を、2013年7月1日付のCOPPA法第312条2項に規定された当社のウェブサイト及びアプリケーションの内部作業のサポートの目的にのみ使用します。私があなたのEメールアドレスを貸し出したり販売したりすることはありません。

#### クッキー

あなたが、私が運営するウェブサイトに訪問する際、あなたのコンピューターにあなたのブラウザーを識別するためクッキーを送信するかもしれません。私たちはクッキーを、ユーザー設定を保存したり使用情報をトラッキングすることにより、私が提供するサービスの質を向上させるため使用します。ほとんどのブラウザーは、クッキーを無効にしたり、クッキーがあなたのコンピューターに送信される際にそのことを警告するよう設定することができます。

#### 第三者のサイトへのリンク

私は第三者のウェブサイトもしくは、私が運営するウェブサイトのサービスへのリンク及び／又は接続を提供するかもしれません。このような第三者のウェブサイトのプライバシー慣行もしくはコンテンツに関しての責任を負いません。
